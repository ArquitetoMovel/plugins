# Modular ai Plugins

## Objetivo

O repositório **markV AI Plugins (markv-ai-plugin)** é um repositório de plugins de **IA** no qual oferece conjunto completo de *Workflows* com suporte a desenvolvimento com agentes de IA.
Cada *plugin de AI* criado no repositório deve ser compatível com as seguintes ferramentas

- Claude Code
- Github Copilot
- Copilot CLI
- Cursor
